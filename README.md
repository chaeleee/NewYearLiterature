# 기능 정의

## 공모전 제출 페이지
공모전 접수를 받는 기간 동안 제공될 페이지.

### 공모전 안내
- 예시 작품 전시
- 공모전 안내: 행사 취지, 수상 안내, 공모작 제출
- 주최자 및 연락처 안내

### 작품 제출
- 내용과 작가명 텍스트 입력
- 사용자 식별을 위한 유저 정보 입력 필요.
- 개인정보이용동의 체크 필수.
- 작품 이미지 다운로드: 방금 제출한 작품을 이미지로 다운로드

### 작품 조회
- 모든 작품 조회: 최신순, 오래된 순, 추천순 정렬 기능
- 내 작품 조회

### 작품 추천
- 추천 제한: 로그인된 상태만 추천 가능.

-----------------------------------

## 공모전 마감 페이지
공모전 기간 마감에 따라 제공될 페이지

### 공모전 마감 안내
- 공모전 마감 안내 문구
- 수상작 발표 일시

### 작품 조회
- 모든 작품 조회: 최신순, 오래된 순, 추천 순 정렬 기능
- 내 작품 조회

### 작품 추천
- 추천 제한: 로그인 상태만 추천 가능

-----------------------------------

## 공모전 수상작 발표 페이지

### 수상작 안내
- 수상작 안내 문구

### 수상작 조회
- 수상작 작품 전시

### 알림 메일 전송
- 수상자들에게 수상 안내 메일 전송


-----------------------------------

## 관리자 페이지
관리자 권한의 유저만 접근 가능. 수상작을 검색, 수정, 수상 선정. 유저 정보 관리.

### 응모 기간 관리
- 응모 기간 설정: 시간에 맞춰 응모 마감 페이지로 전환.
- 수상 발표일 설정: 시간에 맞춰 수상작 페이지로 전환. 

### 수상작 관리
- 전체 조회
- 검색 조회
- 수정
- 수상 등록
- 삭제

### 유저 정보 관리
- 유저 조회
- 검색 조회
- 수정
- 삭제
- 수상 메일 전송
